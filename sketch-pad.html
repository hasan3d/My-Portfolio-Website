Sketch Pad



I have made this Sketch Pad as a part of the project assigned by The Odin Project(TOP).I have wrote jQuery to dynamically create HTML elements and add effects to it such as if a user hovers over the Grid the color of the grid changes so that it can give a meaning to the Sketch Pad.I have also used some CSS to style it.

As a user, You have the option to create new grids by simply specifying width and height of the grids.


Technical highlights

  • Pure hand-coded jQuery
  • Dynamically created HTML markup
